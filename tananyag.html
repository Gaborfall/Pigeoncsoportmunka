Danié feladata:

Készíts egy űrlapot, melynek segítségével üzenetet tudsz küldneni egy adott email címre, úgy hogy a tárgy mezőbe írt szöveg kerüljön a tárgyba, az "üzenet szövege" mezőbe írt pedig az üzenet törzsbe!

1. Hozz létre egy form tag-et melyhez hozzárendelsz egy eseményt a form után írt action-paranccsal. Az esemény jelen esetben az email küldése lesz, melyet az action= után írt mailto:emailcím[amire szeretnéd, hogy az emailt küldjék] parancsal tudsz megadni. Az action az az esemény ami a submit input mezőre kattintáskor fog történni.

2. Ezek után hozd létre a input mezőket és az input mezőkhöz tartozó címkéket az input és a label tag-ekkel. Fontos, hogy minden input elementnek állíts be egy ID-t, és minden label elementnek legyen megadva egy for, melyhez az input element ID-jét rendeld, a következő módon: for="az ehhez tartozó input mező ID-je".

3. A submit gomb létrehozásához az input type legyen submit. Opcionálisan megadhatod a többi input element típusát is, aszerint, hogy milyen jellegű információ kerülhessen bele, bár a mi esetünkben ez nem fontos, mert mindkét input mezőbe szöveget várunk. Az üzenet szövege mezőnél érdemes az input tag-et textarea tag-re változtatni, mert ez alakalmasabb arra, hogy hosszabb üzenetet szerkesszünk, egyrészt sima input mezőben nem lehet új sort nyitni, másrészt a textarea mező méretét a felhasználó tetszőlegesen állíthatja üzenete hosszától függően.

4. Már csak az hiányzik, hogy a tárgy-input box-ba írt szöveg az email küldésénél az email kezelőben a tárgy, az üzenet szövege-input boxba írt szöveg pedig a szövegtörzsbe kerüljön. Ehhez annyit kell tennünk, hogy a tárgy elementhez a "subject", az üzenet szövege elementhez pedig a "body" name-et adjuk meg.
